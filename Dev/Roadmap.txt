# Этап 0: Подготовка (День 0)
**Цель:** Настроить среду, чтобы не тратить время потом.

- [ ] Установить Unity 2022.3 LTS (WebGL модуль обязательно)
- [ ] Создать новый 2D проект, настроить папки: `Scripts/`, `Prefabs/`, `Scenes/`, `Art/`, `Audio/`, `Settings/`
- [ ] Установить DOTween (импорт через Package Manager)
- [ ] Установить Newtonsoft.Json (через Package Manager или .dll)
- [ ] Скачать бесплатные ассеты (Kenney: круги, частицы, звуки — опционально)
- [ ] Создать сцену `MainMenu.unity` и `GameScene.unity`

**Результат:** Готовый проект с правильной структурой.

---

# Этап 1: Ядро игры (Дни 1–5)
**Цель:** Сделать игру, в которую можно поиграть (без прогрессии, без рекламы).

## День 1: Арена и базовое управление
- [ ] Создать круглую арену (спрайт радиусом 10, коллайдер-граница)
- [ ] Создать префаб камня (круг радиусом 0.5, цвет — серый)
- [ ] Написать `PlayerController.cs`:
  - Отслеживание зажатия ЛКМ (OnMouseDown/Up)
  - Визуальная линия направления (LineRenderer)
  - Кружок силы (Scale или спрайт)
  - Расчёт скорости: `speed = clamp(chargeTime * 8, 3, 16)`
- [ ] Написать `CustomPhysicsBody.cs`:
  - velocity, Update → transform.Translate
  - Замедление: `velocity *= 0.995f`
- [ ] Проверить: камень летит после отпускания, тормозит, останавливается у границы

**Результат:** 🎯 Игрок может разгонять камень по арене.

---

## День 2: Столкновения и отскоки
- [ ] Добавить камню `CircleCollider2D` (IsTrigger = true)
- [ ] Написать `CollisionResolver.cs`:
  - OnTriggerEnter2D → определяет, с чем столкнулись (стена, колонна, другой камень)
  - Расчёт нового вектора скорости: `Vector2.Reflect(velocity, normal)`
  - Коэффициент отскока (пока 0.7 для всего)
- [ ] Добавить 2 тестовые колонны (префаб, круг радиусом 0.8)
- [ ] Проверить: камни отскакивают от стен, колонн и друг от друга

**Результат:** Полноценная физика столкновений.

---

## День 3: Боты (простая версия)
- [ ] Создать префаб бота (тот же камень, но другой цвет — серый)
- [ ] Написать `BotAI.cs` (первая версия):
  - Выбор случайной точки на арене каждые 2 секунды
  - Зарядка 0.5–1.5 секунды (визуальный кружок)
  - Разгон в выбранном направлении
- [ ] Добавить 3 ботов на сцену
- [ ] Проверить: боты двигаются, сталкиваются, не вылетают за арену

**Результат:** 🎯 Есть противники, можно играть 1 против 3.

---

## День 4: Урон от угла + здоровье 100 HP
- [ ] Добавить камням `HealthSystem.cs`:
  - maxHealth = 100, currentHealth = 100
  - Событие OnDamageTaken(int damage, Vector2 hitDirection)
  - Событие OnDeath()
- [ ] Написать `DamageCalculator.cs`:
  - Расчёт угла между velocity жертвы и направлением удара
  - damage = 40 (спина), 15–25 (бок), 0 (лоб)
- [ ] При столкновении вызывать `TakeDamage()` у жертвы
- [ ] Проверить: урон правильный, камни умирают при 0 HP

**Результат:** Тактический бой с уроном от угла.

---

## День 5: HP-бары и смерть
- [ ] Создать World Space Canvas как дочерний объект камня
- [ ] Добавить Slider (ширина 2, высота 0.2) — зелёный/красный
- [ ] Привязать значение slider к `currentHealth / maxHealth`
- [ ] Добавить анимацию урона: вспышка спрайта + DOTween для HP-бара (зелёная полоса дёргается)
- [ ] При смерти: анимация осколков (частицы), удаление камня через 0.5 сек
- [ ] Проверить окончание раунда: когда остался 1 камень → текст «Победа!»

**Результат:** 🎯 Полноценный раунд с HP, уроном и победой.

# Этап 2: Контент и прогрессия (Дни 6–12)
**Цель:** Добавить реиграбельность, валюту и систему прогрессии.

## День 6: Два типа колонн

- [ ] Создать префаб упругой колонны (урон 0 HP, отскок 1.2)
- [ ] Расставить на арене: 3 жёстких + 2 упругих (симметрично)
- [ ] Добавить разные звуки для каждого типа (пока заглушки)

- [ ] Реализовать таймер 10 секунд (автовыбор первого)

## День 8: Валюта «Осколки» (базовая)

- [ ] Открытие скина → сохраняется в PlayerPrefs

- [ ] Проверить: смена скина работает

## День 10: Рейтинг и таблица лидеров (локально)

- [ ] Отображать рейтинг и ранг на UI
